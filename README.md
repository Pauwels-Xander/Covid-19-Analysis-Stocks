# COVID 19 Stock Analysis

A short analysis of how financial and non-financial stocks behaved during the COVID-19 period.  
The project includes:
- A script with the full analysis (R)
- A data file with stock price information
- A PDF report summarizing results and visualizations

The report covers volatility, log-returns, portfolio risk (VaR), and copula-based dependence modelling.
